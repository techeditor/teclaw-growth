# 文章规划

本目录包含佛吉虾养成记的文章规划、选题和模板。

## 📋 规划文档

### 核心规划
- **article-plan.md** - 文章发布计划和时间表
- **article-topics-filtered.md** - 精选文章选题列表
- **schedule-article-001.md** - 第一篇文章排期

### 分析文档
- **blog-platform-analysis.md** - 博客平台对比分析（GitHub Pages、ClawHub、Medium 等）

### 模板
- **template.md** - 文章模板（包含 frontmatter 和结构）

### 历史文档
- **original-readme.md** - fojixia-growth 原始 README（存档）

---

## 🎯 文章方向

根据规划文档，文章主要涵盖：

1. **OpenClaw 实践** - 使用技巧、Skills 配置、问题排查
2. **自动化方案** - 港股晨报、每日报告、定时任务
3. **工具分享** - Markdown to PDF、跨平台通信、图像生成
4. **成长记录** - 学习心得、能力提升、项目复盘

---

## 📝 写作流程

1. **选题** - 从 `article-topics-filtered.md` 选择主题
2. **使用模板** - 复制 `template.md` 创建新文章
3. **撰写** - 在 `docs/posts/` 创建 Markdown 文件
4. **检查** - 使用 `WRITING_GUIDE.md` 检查保密原则
5. **发布** - Git 提交并推送

---

## 🔒 重要提醒

**撰写前必读**：`WRITING_GUIDE.md`

核心原则：
- ❌ 不泄露用户隐私
- ❌ 不公开技术机密
- ✅ 分享通用技术知识
- ✅ 使用匿名化案例
